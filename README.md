## ТРЕБОВАНИЯ ПО ОФОРМЛЕНИЮ (Методические рекомендации 2025)

- **Шрифт:** Times New Roman, кегль 12–14
- **Поля:** левое 25 мм, верхнее 20 мм, правое 15 мм, нижнее 20 мм
- **Интервал:** одинарный
- **Нумерация страниц:** арабские цифры, по центру снизу, кегль 10
- **Титульный лист** включается в нумерацию, но номер на нём не ставится
- **Таблицы:** слово "Таблица 1" перед таблицей
- **Рисунки:** "Рисунок 1." по центру под рисунком
- **Ссылки на источники:** в квадратных скобках [1]

### Структура УМК по рекомендациям:
```
Титульный лист
Содержание
Лист согласования
Академический календарь
1. Учебно-методический блок:
   1.1. Теоретическая часть (тема, план, учебная инфо, вопросы для самоконтроля)
   1.2. Практикум (тема, план, задания, указания)
   1.3. Методические рекомендации по изучению
   1.4. Глоссарий
2. Диагностико-контролирующий блок (оценочный лист, тесты)
3. Блок наглядно-дидактического материала (презентации, видео)
```
